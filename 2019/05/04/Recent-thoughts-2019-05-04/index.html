《清华大学王观堂先生纪念碑铭》

士之读书治学，盖将以脱心志于俗谛之桎梏，真理因得以发扬。思想不自由，毋宁死耳。斯古今仁圣所同殉之精义，夫岂庸鄙之敢望。先生以一死见其独立自由之意志，非所论于一人之恩怨、一姓之兴亡。呜呼！树兹石于讲舍，系哀思而不忘。表哲人之奇节，诉真宰之茫茫。来世不可知者也，先生之著述，或有时而不章。先生之学说，或有时而可商。惟此独立之精神，自由之思想，历千万祀，与天壤而同久，共三光而永光。

隋炀帝丝绸缠树

帝以诸蕃酋长毕集洛阳，丁丑，于端门街盛陈百戏，戏场周围五千步，执丝竹者万八千人，声闻数十里，自昏达旦，灯火光烛天地；终月而罢，所费巨万。自是岁以为常。诸蕃请入丰都市交易，帝许之。先命整饰店肆，檐宇如一，盛设帷帐，珍货充积，人物华盛，卖菜者亦藉以龙须席。胡客或过酒食店，悉令邀廷就坐，醉饱而散，不取其直，绐之曰： “中国丰饶，酒食例不取直。”胡客皆惊叹。其黠者颇觉之，见以缯帛缠树，曰：“中国亦有贫者，衣不盖形，何如以此物与之，缠树何为？”市人惭不能答。

《资治通鉴·卷第一百八十一·隋纪·炀皇帝》

证书原因导致Firefox所有扩展失效

https://tieba.baidu.com/p/6120727978

https://bugzilla.mozilla.org/show_bug.cgi?id=1548973

https://bugzilla.mozilla.org/show_bug.cgi?id=1549017

http://tieba.baidu.com/p/6121373774

  1. 地址栏输入about:support，回车，点击打开文件夹打开配置文件夹。
  2. 在配置文件夹里找到extensions.json。
  3. 用Emacs或其他编辑器替换所有的"appDisabled":true为"appDisabled":false，替换所有的"signedState":-1为"signedState":2。
  4. 保存修改，重启Firefox。
  5. 地址栏输入about:addons，回车，进入扩展界面，这时候先禁用所有扩展，再重启启用。
  6. 上述操作暂时让浏览器可以正常使用。

解析：

  1. 这里用的数据是户籍人口，来源于市统计年鉴。
  2. 户籍人口相对有滞后性，不能灵活准确反映人口变化。
  3. 两个「断点」与人口结构(死的多生的少)、经济状况有关。
  4. 政策拉动了人口增长。这方面山东做的不错，全国第一。
  5. 可以预见，未来几年会和2012年的「断点」一样。
