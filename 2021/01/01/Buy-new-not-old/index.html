买新不买旧——玩无线网卡的经历

2021-01-01 | 网络折腾 | 阅读

🖧

请搭配玩路由器的帖子阅读。搭配的路由器是网件R7500v2和斐讯K2P。

无线网卡升级换代史

我现在的笔记本前后用过3块无线网卡：

  1. 第一块网卡是戴尔Inspiron 灵越 17R Turbo带的Intel N 2230，2013年的中端产品，性能实在是垃圾，不支持5 G频段。
  2. 2018年初受不了，在淘宝买了BCM43228HMB，这款网卡似乎是2011年的产品，应该是当时的旗舰，是用在高贵的苹果笔记本上的，总算可以跑满百兆网络了。
  3. 2021年又感觉老网卡到瓶颈了，换了Intel AC 8265，这是2016年的中端产品。最新的所谓WiFi 6网卡需要用Windows 10才支持，我不愿意为了一个网卡换系统，所以不考虑。

测速

用iperf3测速。简单起见，只测了TCP协议下的带宽，用的默认参数。测速服务器在路由器下面，运行iperf3的服务端，服务器和路由有线连接。测速跑了好几次，取最大值，结果如下。

网卡和装法 带宽(Mbits/sec)
Intel N 2230装台式机里 + 垃圾天线 + 放桌子底下 16.5
Intel N 2230装台式机里 + 延长线 + 天线 21.4
Intel N 2230装台式机里 + 不接天线 0.6
Intel N 2230装台式机里 + 垃圾天线 20.0
BCM43228HMB装笔记本里 48.1
BCM43228HMB装台式机里 + 延长线 + 天线 51.2
BCM43228HMB装台式机里 + 不接天线 18.2
Intel AC 8265装笔记本里 139.0

结果分析：

  1. 把网卡接延长线放到信号好的地方，可以提高一点性能，但测了好几次，从最大带宽看，差别不大。可能结果是有显著性差异的，我没具体统计。
  2. 对测得带宽影响最大的是低速的结果，不接天线速度慢，是因为有几秒几乎没有数据，导致最终结果很低。所以信号强弱还是有影响的。
  3. 协议很重要，2.4 G频段的网卡怎么也跑不过5 G频段的。
  4. 买新不买旧。再高端的产品，放几年，也落伍了。

更严谨的测试还可以测ping值，测UDP协议的表现，甚至分析网络数据，我不想深入研究，就不玩了。

心得和收获

  1. 无线协议对无线带宽的影响太大了。
  2. 就像刚才说的，这里只关注带宽其实很片面。
  3. 注意，如果要换网卡，需要先完全删除原来网卡的驱动，再装，否则装不上。
