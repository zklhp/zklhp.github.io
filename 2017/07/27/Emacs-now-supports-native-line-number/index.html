Emacs现在支持原生的行号显示

Jul 27, 2017 | Emacs | 阅读

2017-07-25之后的Emacs开发版支持新的行号显示方式，用M-x global-display-line-numbers-mode或M-x display-line-numbers-mode激活全局和当前buffer的行号显示。

效果如图。

和之前的linum-mode不同的是，这种方式是全局的。感谢Eli Zaretskii的辛苦工作。
